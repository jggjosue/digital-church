# **App Name**: Churchlytics

## Core Features:

- Dashboard Overview: Display key church metrics, activities, and statistics in an easy-to-understand dashboard.
- Member Statistics: Calculate and display metrics for total members and member demographics.
- Attendance Tracking: Monitor weekly attendance numbers and display trends.
- Giving Trends: Track financial giving trends over time and display changes.
- Visitor Analysis: Record new visitor counts and display percentage increases.
- Upcoming Events: List upcoming church events with dates, times, and locations.
- Prayer Request Management: Allow users to submit prayer requests and display recent requests.

## Style Guidelines:

- Primary color: Light blue (#B1D4E0) to evoke trust and serenity.
- Background color: Off-white (#F9FBFC), nearly desaturated blue hue, to provide a clean and modern feel.
- Accent color: Slightly darker blue (#8CB4C2), similar to primary but desaturated, to highlight key data and CTAs without overwhelming the user.
- Body and headline font: 'PT Sans' (sans-serif) for a modern and approachable look, suitable for both titles and longer texts.
- Use simple, consistent icons to represent different metrics and actions.
- Design a card-based layout for clear data segmentation and readability.
- Employ subtle transitions and animations for visual feedback upon data updates.
